# Bela Climb 🐐

Jogo de plataforma vertical (HTML5 + Canvas + JavaScript, sem dependências).

## Jogar
Publicado via GitHub Pages — o link fica em Settings > Pages do repositório.

## Estrutura
- `index.html` — página principal
- `css/style.css` — estilos
- `js/` — módulos do jogo (config, audio, levels, physics, hazards, render, input, game)
